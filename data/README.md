# Data Directory

This folder stores datasets used in the Inside Airbnb Market Intelligence Platform.

## Structure

- `raw/`  
  Original Inside Airbnb source files. These files are not committed to Git because they can be large and can be downloaded again.

- `interim/`  
  Temporary intermediate files created during processing. This folder is reserved for future pipeline stages.

- `processed/`  
  Cleaned and transformed datasets such as calendar metrics, review metrics, and final listing master tables. These files are generated by the pipeline and are not committed to Git.

- `external/`  
  External enrichment datasets, if used in future versions of the project. This folder is currently reserved.

## Reproducibility

Processed datasets can be recreated by running:

```bash
python -m src.pipeline_runner
```
