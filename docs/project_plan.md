# Project Plan

## Project Name

Inside Airbnb Market Intelligence Platform — Data Engineering, Analytics & AI

## Project Goal

The goal of this project is to build an end-to-end market intelligence platform using Inside Airbnb data for Amsterdam.

The project transforms raw Airbnb datasets into clean analytical outputs, builds a DuckDB warehouse, performs business analysis, applies statistical testing, and trains a machine learning model for price prediction.

## Scope

This project focuses on:

- Raw data ingestion
- Dataset profiling
- Data quality validation
- Data cleaning
- Feature engineering
- Calendar availability aggregation
- Review activity aggregation
- Final listing master creation
- DuckDB warehouse design
- SQL analytical queries
- Exploratory data analysis
- Statistical analysis
- Machine learning price prediction
- Model explainability
- Responsible AI usage documentation
- Open innovation concept
